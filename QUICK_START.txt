🔧 TROUBLESHOOTING:

Problem: CSS not loading
Solution: Check file path in HTML, ensure css/style.css exists

Problem: JavaScript errors
Solution: Open browser console (F12), check for errors

Problem: Python not found
Solution: Install Python from python.org, add to PATH

Problem: Can't activate venv
Solution: Run: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

Problem: TensorFlow won't install
Solution: Try: pip install tensorflow==2.13.0
